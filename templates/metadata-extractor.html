Metadata Extractor

A powerful solution for seamlessly compiling and merging multiple keyhole markup language files into a single, cohesive dataset. Streamlining the integration process, this tool ensures efficiency and accuracy in geospatial data management, catering to users seeking unified and consolidated KML file outputs.

Instructions

  1. 1. Click Choose Files and select all the KML files you want to merge;
  2. 2. Once the KML files have been selected, click Upload;
  3. 3. Wait for the merge process to complete successfully;
  4. 4. Save the new consolidated KML file by clicking Download;
  5. 5. If you wish to remerge KML files, click New Upload.
